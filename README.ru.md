# 💥 Comic Builder

**Комиксы про айти в четыре панели — без умения рисовать.** Работает в браузере, бесплатно, без бэкенда.

🇬🇧 [English version](README.md) · **[Попробовать →](https://termyanen.github.io/comic-builder/)**

![Пример стрипа](landing/examples/example-2.png)

## Что внутри

- **12 персонажей** — дев, робот-ИИ, сеньор, ПМ, босс, тестировщик, дизайнер, девопс, стажёр, клиент, HR и кот. 9 поз, 8 эмоций, направление взгляда, полуоборот тела, предметы в руках, своя одежда на серию.
- **Язык комиксов** — бабблы с голосами (крик, мысль, шёпот, робо), окно терминала с чипами команд, жёлтые кнопки подтверждения, крупный леттеринг «БАМ!», эффекты (удар, пот, zzz, фокус-линии).
- **Сценарий → стрип** — пишешь текст (`герой злой: реплика`), стрип собирается сам. Понимает русский и английский.
- **Всё перетаскивается** — персонажи, бабблы, хвосты бабблов, эффекты и подписи двигаются мышью прямо в панели. Undo/redo, стрелки для точной доводки, копирование и обмен панелей, шаблоны сцен (встроенные + свои).
- **Экспорт под площадки** — квадрат 1080×1080 (Instagram/Threads), вертикальная лента 1×4 (Telegram), карусель по панелям — или всё разом одной кнопкой. Шрифт встраивается в экспорт, серии сохраняются в JSON.
- **Без бэкенда** — всё хранится в localStorage браузера.

## Быстрый старт

```bash
npm install
npm run dev
```

Node 22+. Продакшен-сборка: `npm run build` (статика в `dist/`).

## Формат текстового сценария

```
название: Стендап
номер: #002

панель 1
фон: офис, тема песок
герой довольный печатает: Всё, фича готова!
робот: Проверить перед деплоем?

панель 2
терминал: Проверяю... | Найдено проблем: 0 | $ ship-it
```

Каждая панель начинается с `панель N`. Реплика — `кто [настроение] [поза]: текст`: первый упомянутый персонаж встаёт справа, второй — слева, хвосты сами указывают на говорящего. Спецстроки: `фон`, `тема`, `подпись`, `эффект`, `терминал` (строки через `|`), `кнопка`.

## Об архитектуре

- React + TypeScript + Vite + Zustand, рендер — чистый SVG (без канваса).
- Персонаж — это данные, а не картинка: позы — пути конечностей, эмоции — слой лица, одежда — цвета. Новый персонаж — файл на ~10 строк в `src/svg/characters/`.
- PNG-экспорт сериализует SVG со встроенным base64-шрифтом — иначе растеризатор молча подменит его системным.

## Авторы

Визуальный язык вдохновлён классической традицией минималистичных веб-комиксов. Шрифт: [Balsamiq Sans](https://fonts.google.com/specimen/Balsamiq+Sans) (OFL).
